# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please DO NOT open a public issue.

Instead, please report it privately to the maintainer to give us time to patch the issue before making it public.

- Email the maintainer directly or use GitHub's private vulnerability reporting feature.
- Provide a detailed summary of the vulnerability, including step-by-step instructions on how to reproduce it.
